# 04 数据库设计

## 4.1 数据库说明

- 数据库类型：`MySQL`
- 字符集建议：`utf8mb4`
- 命名规范：表名与字段名使用下划线风格

## 4.2 核心表清单

- `users`：用户表（含管理员与普通用户）
- `books`：图书表
- `borrow_records`：借阅记录表
- `reservations`：预约记录表
- `import_jobs`：导入任务记录表（可选）
- `export_jobs`：导出任务记录表（可选）

## 4.3 表结构草案

## 4.3.1 users

- `id` BIGINT PK
- `username` VARCHAR(64) UNIQUE NOT NULL
- `password_hash` VARCHAR(255) NOT NULL
- `role` ENUM('ADMIN','USER') NOT NULL
- `real_name` VARCHAR(64) NULL
- `phone` VARCHAR(20) NULL
- `email` VARCHAR(128) NULL
- `status` TINYINT NOT NULL DEFAULT 1
- `created_at` DATETIME NOT NULL
- `updated_at` DATETIME NOT NULL

## 4.3.2 books

- `id` BIGINT PK
- `isbn` VARCHAR(32) UNIQUE NULL
- `title` VARCHAR(255) NOT NULL
- `author` VARCHAR(128) NOT NULL
- `category` VARCHAR(64) NULL
- `publisher` VARCHAR(128) NULL
- `publish_date` DATE NULL
- `total_stock` INT NOT NULL DEFAULT 0
- `available_stock` INT NOT NULL DEFAULT 0
- `status` TINYINT NOT NULL DEFAULT 1
- `created_at` DATETIME NOT NULL
- `updated_at` DATETIME NOT NULL

## 4.3.3 borrow_records

- `id` BIGINT PK
- `user_id` BIGINT NOT NULL FK -> users.id
- `book_id` BIGINT NOT NULL FK -> books.id
- `borrowed_at` DATETIME NOT NULL
- `due_at` DATETIME NOT NULL
- `returned_at` DATETIME NULL
- `status` ENUM('BORROWED','OVERDUE','RETURNED') NOT NULL
- `created_at` DATETIME NOT NULL
- `updated_at` DATETIME NOT NULL

## 4.3.4 reservations

- `id` BIGINT PK
- `user_id` BIGINT NOT NULL FK -> users.id
- `book_id` BIGINT NOT NULL FK -> books.id
- `status` ENUM('ACTIVE','CANCELLED','FULFILLED','EXPIRED') NOT NULL
- `reserved_at` DATETIME NOT NULL
- `expired_at` DATETIME NULL
- `created_at` DATETIME NOT NULL
- `updated_at` DATETIME NOT NULL

## 4.4 索引建议

- `users(username)` 唯一索引
- `books(title)`、`books(author)` 普通索引
- `borrow_records(user_id,status)` 联合索引
- `borrow_records(book_id,status)` 联合索引
- `reservations(user_id,status)` 联合索引
- `reservations(book_id,status)` 联合索引

## 4.5 数据一致性约束

- 借阅成功必须确保 `available_stock > 0`
- 借阅成功后 `available_stock` 原子减 1
- 归还成功后 `available_stock` 原子加 1
- 单用户借阅中的记录数不得超过 5
- 逾期用户默认禁止新增借阅
